## Start Project 
```console
git clone https://github.com/kaudx-hub/laravel_vue_apollo.git
```
```console 
composer install
``` 
```console
npm install
```
```console 
cp .env.example .env 
``` 
```console 
php artisan key:generate
``` 

Connect database on .env
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=Name
DB_USERNAME=root
DB_PASSWORD=
```

```console 
php artisan migrate
```
```console
php artisan db:seed
```
```console 
php artisan serve
```
```console
npm run watch
```
